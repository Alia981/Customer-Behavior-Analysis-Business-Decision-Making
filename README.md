# Customer Behavior Analysis

A real-world statistics project that analyzes mall customer data to understand customer demographics, income levels, and spending behavior for better business decision-making.

## Project Overview

This project explores a mall customer dataset containing information such as Customer ID, Gender, Age, Annual Income, and Spending Score. The main aim is to study customer patterns and identify how different groups behave so that businesses can plan marketing strategies and customer segmentation more effectively.

## Objectives

- Understand the structure of the customer dataset.
- Check data quality by identifying missing values and data types.
- Perform descriptive statistical analysis.
- Visualize customer behavior using histograms and boxplots.
- Interpret the results from a business perspective.

## Dataset Description

The dataset contains 200 customer records with 5 columns:

- **CustomerID**: Unique identifier for each customer.
- **Gender**: Customer gender.
- **Age**: Age of the customer.
- **Annual Income (k$)**: Annual income in thousands of dollars.
- **Spending Score (1-100)**: Score assigned based on customer spending behavior.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Data Understanding

The first part of the project focuses on understanding the dataset:
- Displaying the dataset shape.
- Listing all columns.
- Checking for missing values.
- Reviewing data types.

This step confirms that the dataset is clean and suitable for analysis, with no missing values.

## Descriptive Analysis

The project calculates:
- Mean
- Median
- Standard deviation

for:
- Age
- Annual Income
- Spending Score

### Key Findings

- Annual Income shows the highest variability.
- Customers are diverse in income and spending patterns.
- Age is comparatively less variable than income and spending score.

## Visualizations

The project uses graphs to better understand the data:
- **Histograms** to study the distribution of Age, Income, and Spending Score.
- **Boxplots** to identify spread and possible outliers.

These visualizations help in understanding how customer values are distributed across the dataset.

## Business Insights

From the analysis, we can conclude that:
- Customers are not all similar; they belong to different spending groups.
- Income and spending score can help in customer segmentation.
- Businesses can target customers more effectively using this information.
- The analysis can support marketing and sales planning.

## Conclusion

This project successfully analyzes mall customer data using statistical and visual methods. It helps in understanding customer diversity and demonstrates how data analysis can support business decisions. The project is a good example of using Python for exploratory data analysis in a real-world scenario.
